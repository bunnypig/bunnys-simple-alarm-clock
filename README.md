# Bunny's Simple Alarm Clock
A simple alarm clock for Linux KDE Wayland

This is an alarm clock for Linux KDE with Wayland support and it supports custom audio files.
I made this with help of AI and it's made for personal use mainly. I might update it in the future with new features or just update it if it breaks.
I've used the MIT License so feel free to do whatever you want with the code.

How to install:

1. Add the COPR repository

   dnf copr enable bunnypig/bunnys-simple-alarm-clock

2. Install the program

   dnf install bunnys-simple-alarm-clock
